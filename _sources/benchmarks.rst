.. _benchmarks:

Benchmarks
==========

This section contains various benchmarks comparing mp++ to other multiprecision libraries. All benchmarks
were run on an AMD Ryzen 3950x in a 64-bit GNU/Linux environment, using the GCC compiler.
The benchmarking code is available `here <https://github.com/bluescarni/mppp/tree/master/benchmark>`__.

The benchmark results were last updated on **20200609**, using the following package versions:

* GCC 10.1,
* mp++ 0.21,
* GMP 6.2.0,
* Boost 1.73.0,
* FLINT 2.5.2.
